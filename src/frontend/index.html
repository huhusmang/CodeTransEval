🚀💻 代码生成模型的功能性和安全性评估系统 🔒🔍

简介

欢迎使用代码生成模型的功能性和安全性评估系统，这是一个用于评估代码生成模型在代码翻译任务上的工具。 在机器学习和人工智能不断发展的环境下，准确和安全地在不同编程语言之间翻译代码至关重要。 我们的项目通过提供一个全面、自动化的解决方案来评估代码生成模型的的有效性和安全性，从而满足这一需求。

主要功能

  • 自动获取模型输出

    代码翻译评估器的核心功能是能够自动从各种代码翻译模型中获取输出。这一特性确保了无缝集成，并提供了一种无需手动操作即可获取模型输出进行评估的方法。

  • 标准化代码格式

    考虑到代码结构和语法的多样性，我们的工具能够熟练地提取这些模型输出，并将其转换为标准化的代码格式。这种统一性对于在不同模型之间进行准确评估和比较至关重要。

  • 强大的评估机制

    我们项目的核心在于其强大的评估引擎。它细致地评估翻译后的代码，考虑了准确性和安全性。这个自动化过程不仅节省了时间，而且在评估中引入了精确性和一致性。

  • 全面的评估指标

    理解代码翻译的多方面特性，代码翻译评估器提供了多个评估指标。这些指标从语法正确性、语义准确性和实际应用性等方面全面展示了模型的性能。

  • 便捷的可视化分析

    为了更直观地呈现评估结果，我们的工具提供了丰富的可视化功能。通过生成清晰易懂的图表和图形，用户可以快速洞察模型的表现，识别优势和改进领域。这种可视化分析使得评估结果的解释和交流更加便捷高效。

本地部署要求

需要安装的工具：

Python

版本要求：3.10

OpenJDK

版本要求：17 (Java)

C++

版本要求：20

安装所有必需的 Python 依赖项：

  • dashscope
  • numpy
  • codebleu
  • qianfan
  • Requests
  • volcengine
  • zhipuai

安装所有必需的 Python 依赖项（如果您之前已经设置过依赖项且版本要求不严格，可以跳过此步骤）：

pip install -r requirements.txt
